# QuizMind 技术操作手册

这份手册面向开发者、部署者和深度使用者，重点覆盖安装、配置、运行、数据目录、主流程、排障和维护建议。

## 1. 环境要求

- Python `3.10+`
- Windows / macOS / Linux
- 可访问兼容 OpenAI 协议的模型服务
- 推荐使用独立虚拟环境

## 2. 安装步骤

```powershell
cd d:\Code\Python\QuizMind
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
```

## 3. 启动应用

```powershell
streamlit run app.py
```

默认访问地址通常是：

```text
http://localhost:8501
```

## 4. 最小配置

建议至少在 `.env` 中配置：

```env
SILICONFLOW_API_KEY=
SILICONFLOW_BASE_URL=https://api.siliconflow.cn/v1
SILICONFLOW_MODEL=deepseek-ai/DeepSeek-V3.2
```

如果还要启用远程 embedding，可补充：

```env
SILICONFLOW_EMBEDDING_MODEL=
```

## 5. 常用运行参数

### AI 生成相关

- `QUIZMIND_STRICT_AI_GENERATION`
  - `true`：严格模式，AI 失败不自动降级
  - `false`：兼容模式，AI 失败时允许本地规则回退
- `QUIZMIND_FAST_MODE`
  - 更偏速度
- `QUIZMIND_QUALITY_FIRST`
  - 更偏质量
- `QUIZMIND_AI_QUALITY_REVIEW`
  - 是否启用 AI 质量审查
- `QUIZMIND_AI_QUALITY_THRESHOLD`
  - 题目质量阈值

### 模型调用相关

- `QUIZMIND_LLM_TIMEOUT`
- `QUIZMIND_LLM_MAX_RETRIES`
- `QUIZMIND_QUIZ_GEN_ATTEMPTS`
- `QUIZMIND_QUIZ_GEN_CONCURRENCY`
- `QUIZMIND_MAX_OUTPUT_TOKENS`

### 内容压缩与摘要相关

- `QUIZMIND_SOURCE_CHAR_LIMIT`
- `QUIZMIND_SOURCE_SUMMARY_CHUNK_SIZE`
- `QUIZMIND_SOURCE_SUMMARY_MAX_CHUNKS`
- `QUIZMIND_SOURCE_SUMMARY_CONCURRENCY`
- `QUIZMIND_QUIZ_POINTS_LIMIT`
- `QUIZMIND_QUIZ_SEGMENTS_LIMIT`

## 6. 运行时目录

应用运行后会自动生成 `.quizmind_runtime/`：

- `.quizmind_runtime/memory/`
  - 记忆库和向量索引
- `.quizmind_runtime/quiz_bank/`
  - 已保存题库和索引
- `.quizmind_runtime/user_data/features.json`
  - 收藏、学习记录、阶段轨迹、恢复上下文
- `.quizmind_runtime/quizmind.log`
  - 运行日志

## 7. 当前主流程

## 7.1 当前内容一键出题

1. 进入 `输入与解析`
2. 选择输入方式
3. 提供文本、文件或 URL
4. 点击 `一键解析并出题`
5. 进入作答区开始训练

系统会自动执行：

- 内容解析
- 题目生成
- 必要时质量修复
- 记忆库存档
- 进入作答区

## 7.2 记忆模式一键出题

1. 先确保记忆库里已有内容
2. 侧边栏切换到 `记忆模式`
3. 输入检索词
4. 点击 `一键从记忆出题`

适合：

- 复习某一知识域
- 刷记忆库中沉淀过的主题
- 短时间快速开练

## 7.3 每日训练

每日训练会结合：

- 当前学习阶段
- 当前报告中的强化知识点
- 历史薄弱点
- 自适应难度建议

特点：

- 不是固定模板题单
- 题量、题型和难度会被学习阶段影响

## 7.4 智能续练

提交结果后，系统会根据错因和阶段进入下一轮训练。  
当前已经支持：

- 阶段识别
- 阶段推进轨迹
- 阶段化复盘
- 阶段化作答引导

## 7.5 继续上次训练

系统会把未完成训练的 `parsed + quiz + 来源信息` 存到用户数据里。  
首页可直接恢复到上次训练状态。

## 7.6 场景模拟

场景模拟支持：

- 模板快速填充
- 引导模式
- 严格模式
- 多轮追问
- 结果评分与建议

## 8. 学习逻辑说明

当前版本内置 `QuizMind CORE 学习法`，阶段包括：

- `理解建图`
- `主动回忆`
- `推理纠偏`
- `细节校准`
- `表达固化`
- `综合进阶`

这套学习法会影响：

- 每日训练聚焦点
- 智能续练生成
- 结果页复盘重点
- 作答区提示文案
- 阶段推进轨迹

## 11. 常见问题排查

### 11.1 记忆模式没有内容

原因：

- 还没有把任何学习内容保存到记忆库

处理：

1. 回到当前内容模式
2. 使用 `保存到记忆库`
3. 再切回记忆模式

### 11.2 AI 出题失败

排查顺序：

1. 检查 `SILICONFLOW_API_KEY`
2. 检查 `SILICONFLOW_BASE_URL`
3. 检查 `SILICONFLOW_MODEL`
4. 看是否开启了严格模式

说明：

- 严格模式下，AI 失败不会自动回退到本地规则

### 11.3 URL 抓取失败

可能原因：

- 目标站点限制抓取
- 当前网络无法访问
- 页面内容过于动态

建议：

- 先手动复制正文内容再粘贴导入

### 11.4 PDF 导出中文异常

建议优先：

- 导出 Markdown
- 导出 JSON

截至当前版本：

- 主流程可编译、可导入

- 生成质量仍受模型服务和输入材料质量影响
