# QuizMind 操作步骤（详细版）

本文档按“从零到可用”方式编写，直接照做即可跑通。

## 1. 初始化项目

1. 打开终端，进入项目根目录：

```powershell
cd d:\Code\Python\QuizMind
```

2. 创建并激活虚拟环境：

```powershell
python -m venv .venv
.venv\Scripts\activate
```

3. 安装依赖：

```powershell
pip install -r requirements.txt
```

4. 复制环境变量模板：

```powershell
copy .env.example .env
```

5. 编辑 `.env`，至少填写：

- `SILICONFLOW_API_KEY`
- `SILICONFLOW_BASE_URL`（默认即可）
- `SILICONFLOW_MODEL`（默认即可）

## 2. 启动应用

运行：

```powershell
streamlit run app.py
```

浏览器打开终端输出的本地地址（通常是 `http://localhost:8501`）。

## 3. 首次进入后的建议设置

左侧边栏建议依次设置：

1. `模型与 API 设置`
   - 若使用 `.env`，保持“使用默认 .env”并点击“应用设置”。
   - 若临时改模型，切到“使用自定义（已保存）”，填写后应用。
2. `练习配置`
   - 内容来源：先选“当前内容”。
   - 作答模式：首次建议“练习模式”。
3. `题目参数`
   - 题目数量先用 5~10。
   - 难度比例建议 `30/50/20`（简单/中等/困难）。
   - 题型比例按你的学习目标调整。

## 4. 智能练习完整流程

## 4.1 选择输入方式

在“智能练习”工作区，选择一种输入方式：

- 粘贴文本：直接贴笔记/讲义
- 上传文件：支持 `pdf/docx/md/txt`，可多文件
- 网页 URL：输入文章链接

上传文件时可选：

- 综合生成（合并多个文件）
- 单文件生成（从列表选择一个文件）

## 4.2 解析与生成题目

推荐直接点击“一键解析并生成题目”。

系统会自动完成：

1. 内容解析
2. 题目生成
3. 进入作答区

如果你只想分步操作，也可以先解析再点“生成题目”。

## 4.3 作答

系统会按题型自动渲染控件：

- 单选/判断：单选按钮
- 多选：多选框
- 填空/简答：文本输入

练习模式下可随时提交；考试模式下有倒计时，到时自动交卷。

## 4.4 查看评测结果

提交后在“结果复盘”查看：

- 总分
- 客观题正确率
- 主观题平均分
- 错题清单
- 知识点掌握状态
- 复习建议与强化训练题

## 4.5 强化训练

在结果区点击“加载强化题到作答区”，可进入针对薄弱点的强化练习。

## 5. 场景模拟流程（工程面试）

切到顶部工作区：`场景模拟`。

1. 选择模式
   - 引导模式：适合训练阶段
   - 严苛模式：适合冲刺阶段
2. 填写场景描述
   - 可使用模板一键填充后再修改
3. 点击“开始模拟”
4. 在聊天框持续回答 AI 追问
5. 结束后查看结果
   - 分数
   - 是否通过
   - 优势/薄弱点/建议动作

## 6. 功能管理区使用方法

页面底部有 3 个标签页：

1. `题库与队列`
   - 查看历史题目
   - 按条件检索
   - 删除记录
   - 导出 JSON / Markdown / PDF
   - 使用批量队列处理多条内容
2. `收藏与看板`
   - 收藏题目
   - 生成收藏题单
   - 查看学习看板
3. `系统日志`
   - 查看最近运行日志，排查问题

## 7. 记忆模式使用方法

1. 先在“当前内容”模式下完成一次解析/生成。
2. 保存内容到记忆库。
3. 切换边栏“内容来源”为“记忆模式”。
4. 输入检索词并设置召回片段数。
5. 点击“生成题目”。

## 8. 常见问题排查

1. `.doc` 文件无法上传
   - 只支持 `.docx`，请先转换。
2. 模型调用失败
   - 检查 API Key、Base URL、Model 是否正确。
   - 可在“模型与 API 设置”里重新应用。
3. 记忆模式报空
   - 说明尚未保存过记忆快照，先完成一次解析并保存。
4. PDF 导出中文异常
   - 当前使用 `STSong-Light` 字体，若系统环境异常可尝试改为导出 Markdown/JSON。

## 9. 推荐实践

1. 初次学习：练习模式 + 引导模式场景模拟。
2. 考前冲刺：考试模式 + 严苛模式场景模拟。
3. 每次完成后：查看“薄弱知识点”，立即做强化题。
4. 每周复盘：用“收藏与看板”观察成绩趋势并调整难度比例。
